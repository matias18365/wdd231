OS Comparison Hub

Site Purpose

The website aims to educate users about the differences between Windows, macOS, and Linux by providing in-depth comparisons of their user interfaces, performance metrics, security features, and software availability. The information will help users make informed decisions tailored to their personal or professional needs.

Scenario Questions

  • Question 1: What operating system aligns best with my specific needs as a developer or casual user?
  • Question 2: How do the security features of each operating system compare, and which one offers the best protection for my data?

Color Scheme

Colors Defined:

  • Blue (#000088): Used for headers and call-to-action buttons to create a professional and calming effect.
  • White (#FFFFFF): Used for the background and text areas to ensure legibility and a clean aesthetic.

Font Selection

Font: 'Roboto', serif

Application: 'Roboto' will be used for all body text and subheadings in the site plan document for a clean and modern appearance. Headings can be bolded for emphasis.

Home Page Wireframe

Mobile View Wireframe: A vertical layout with a logo at the top, navigation menu below, followed by sections for OS features, pros and cons, and a call-to-action button.

Wider View Wireframe: A horizontal layout with a header section for the logo and menu, a main content area divided into three columns for each OS.

Conclusion

This site plan outlines the essential components of the OS Comparison Hub. It aims to provide valuable information to users to help them make informed decisions regarding their choice of operating system.
